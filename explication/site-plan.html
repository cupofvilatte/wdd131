Explication Site Plan

Vilate Knapp

WDD 131-3

Overview

Purpose

To teach people about explication and to help users develop their explication skills.

Audience

The intended users for this page are primarily philosophy students, though the professors are also considered in the development.

Dynamic elements

One one subpage, Javascript will be used to allow users to click a step of explication and the explanation will drop down below the term. On my second subpage, there will be many examples of explication the user may look at. These will be stored in javascript as an object and also sorted/filtered using javascript. On the third subpage, users will have the ability to try out explication some of their own terms. Javascript will be used to store their practices using local memory, and to provide a random term for the user to explicate.
